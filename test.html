$ Hooks


+ React Hooks are functions that let you "hook into" React state and lifecycle features from function components. They were introduced in React 16.8 to allow function components to manage state and side effects, which was only possible in class components before. Hooks provide a more direct API to the React concepts you already know: props, state, context, refs, and lifecycle. Here are some of the most commonly used React Hooks:


  1. -useState
  2. -useEffect
  3. -useContext
  4. -useReducer
  5. -useCallback
